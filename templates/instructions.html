Instructions

  1. Login to this website and then go to your dashboard
  2. Pick a server you would like to use CommunityPlaylist on (you must be an admin)
  3. Under Bot Status click "Add to Server" button. Refresh and make sure Bot Status is now green
  4. Under Spotify Status click "Add Spotify Account", ensure Spotify Status is now green
  5. Go to the discord server and run these commands:
  6. !set_playlist <playlist name> Choose a playlist to add to. If one already exists on your Spotify account it will use it, otherwise it will create it
  7. !set_channel <Text channelID number> Choose which text channel to monitor for spotify links. You can get your channel id following this guide
  8. !enable Enable the bot. By default it is disabled. This will toggle bot from actively watching for spotify links or not
  9. !get_playlist Use this command to check which playlist the bot is adding to
