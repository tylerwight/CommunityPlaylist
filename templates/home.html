SpotWatcher

This bot is in Under Construction. While you can add it to your discord server, you need to be invited by me to allow it to talk to Spotify.

SpotWatcher is a Discord bot that monitors a Discord Text Channel for spotify links. If it sees one it will add it to a Spotify playlist, allowing you to create a running playlist of all the music you share with each other.

To get started press the button below to add the bot to your discord server.

Add bot

Discord

This bot connects to a discord server

Spotify

This bot authenticates with one spotify user per server

Playlist

This bot will add any spotify song it sees in a given text channel, and add it to a playlist
