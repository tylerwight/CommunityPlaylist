How To Use SpotWatcher

  1. Ensure the bot has been added to your server. You can add it with the button on the home page

  2. Use the "_auth_me <spotify username>" command to authenticate to spotify.

    • Get the username by logging into Spotify on a web browser and going to account section, it could be a random bunch of numbers/letters or a regular username
    • Click the link the bot posts to open spotify in a browser where you are logged into the account you put in the auth_me command
    • Click authorize, it will redirect to this website to finish your authorization

  3. Use "_set_playlist <name>" to choose a Spotify playlist to put stuff in. If it exists in your account already it will add to it, if it doesn't it will create it.

  4. Use "_set_channel <channel_id>" to choose which channel it watches for Spotify links. You can get the text channel id by right clicking -> copy link. Paste the link somewhere to look at it. The channel ID is the final (right-most) number.

  5. Use the "_enable" command to turn on (or off) monitoring for spotify links. If this is off the bot won't react or make playlists. "_enable"

  6. Paste a Spotify link in the channel you set. If all is well, it should be automatically added to the playlist you setup, using the account you authenticated with.
