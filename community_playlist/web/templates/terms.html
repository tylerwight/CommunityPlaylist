2. Bot Functionality

The Bot monitors text channels where it is authorized and listens for Spotify track links. When a link is detected, the Bot adds the track to a predefined Spotify playlist.

3. User Data

  • Message content (to detect Spotify links)

  • Spotify link metadata
  • Encrptyed Spotify Token (for interfacing with Spotify API)

Privacy Policy

Last Updated: June 27, 2025

1. Information Collected

  • Message Content —only messages containing Spotify links are processed.
  • User & Channel Metadata —user ID and channel ID may be logged.
  • Spotify Metadata —track name, artist, album (via Spotify API).
  • Spotify Token  — For creating playlists on behalf of a user (via Spotify API).

2. How We Use Information

To detect Spotify links, add tracks to the playlist, and optionally log additions to prevent duplicates or abuse.

3. Data Retention

No message content or user data is stored permanently. You can choose to delete your Spotify Token and any point via the website.
